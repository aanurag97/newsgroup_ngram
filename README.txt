The 20_Newsgroup is a dataset of 20 classes of newspaper clippings.
The n-gram is defined as 'n' number of words in succession ignoring 
all non-alphanumeric characters.

The python file return top unique n-gram for the whole dataset.

Input format for Unix environment is as follows -

python3 main.py *path* *t* *n* *k*

path - local 20_Newsgroup dataset path
t - number of threads
n - n-gram
k - top k unique outputs

The compressed dataset is available in repository. Link to dataset is -

https://archive.ics.uci.edu/ml/machine-learning-databases/20newsgroups-mld/20_newsgroups.tar.gz
